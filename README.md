Prerequisites
=============

* source-highlight 3.1
* Berkley-DB
* perl
  * Dancer
  * KiokuDB
  * perl bindings for source-highlight:  
    https://code.launchpad.net/~chust/+junk/SourceHighlight
  * Template::Toolkit

Installation
============

    cpan Dancer KiokuDB::Backend::BDB Template

Running
=======

    ./Pastedance.pl
